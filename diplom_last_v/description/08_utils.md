## Код с объяснениями

### `chunk_text()` — разбивка длинного текста

```python
def chunk_text(text: str, limit: int = 3800) -> list[str]:
```
Telegram не принимает сообщения длиннее **4096 символов**. LLM-ответы часто длиннее. Эта функция режет текст на куски ≤3800 символов (с запасом).

```python
    if len(text) <= limit:
        return [text]
```
Если текст короткий — возвращаем как есть (список из одного элемента).

```python
    parts: list[str] = []
    buf = ""
    for block in text.split("\n\n"):
```
Делим текст по **двойному переносу строки** (`\n\n`) — это абзацы. Лучше резать между абзацами, чем посередине предложения.

```python
        if len(buf) + len(block) + 2 > limit:
            if buf:
                parts.append(buf.strip())
            # если блок сам больше limit — режем жёстко
            while len(block) > limit:
                parts.append(block[:limit])
                block = block[limit:]
            buf = block + "\n\n"
        else:
            buf += block + "\n\n"
```
Набираем буфер `buf`. Когда добавление следующего абзаца превысит лимит:
- Сохраняем накопленное в `parts`
- Если сам абзац больше лимита — режем жёстко по `limit` символов
- Начинаем новый буфер

```python
    if buf.strip():
        parts.append(buf.strip())
    return parts
```
Добавляем остаток буфера.

**Пример использования:**
```python
for part in chunk_text(llm_response):
    await message.answer(part)
```

### `money()` — форматирование суммы

```python
def money(amount_minor: int, currency: str = "RUB") -> str:
    """Отформатировать сумму из минимальных единиц в рубли/доллары."""
    major = amount_minor / 100
    sign = "₽" if currency == "RUB" else currency
    return f"{major:,.2f} {sign}".replace(",", " ")
```
Telegram Payments хранит суммы в **копейках/центах** (минимальные единицы). 99900 копеек = 999 рублей.

- `amount_minor / 100` → переводим копейки в рубли
- `f"{major:,.2f}"` → форматируем с двумя знаками после запятой и разделителями тысяч
- `.replace(",", " ")` → заменяем американский разделитель `,` на пробел (русский стиль: `1 000.00 ₽`)

**Пример:** `money(99900, "RUB")` → `"999.00 ₽"`

### `join_lines()` — склейка строк

```python
def join_lines(lines: Iterable[str]) -> str:
    return "\n".join(l for l in lines if l is not None)
```
Склеивает список строк через перенос, пропуская `None`. Используется при форматировании профиля пользователя.

## Зачем выносить в отдельный файл

Представь: `chunk_text()` используется в `agent_chat.py`, `athlete.py` и потенциально в других. Если бы она была написана в каждом файле — при изменении логики нужно было бы менять в нескольких местах. Вынесли в `utils.py` — меняем один раз.
