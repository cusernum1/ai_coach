# app/core/prompts.py — Промпт-шаблоны для LLM

## За что отвечает файл

Здесь хранятся все «инструкции для ИИ» — шаблоны текстов, которые отправляются LLM для генерации планов, анализа и рекомендаций. Каждая функция принимает данные о спортсмене и формирует готовый текст запроса к модели.

Вынесено в отдельный файл — чтобы не смешивать «логику бизнеса» с «текстами инструкций».

## Код с объяснениями

### Вспомогательная функция `_coach_block()`

```python
def _coach_block(base_program: Optional[str]) -> str:
    if not base_program:
        return ""
    return (
        "\n\nБАЗОВАЯ ПРОГРАММА ТРЕНЕРА (учитывай как основу):\n"
        f"{base_program.strip()}\n"
    )
```
Если тренер задал базовую программу (через `/settings`) — встраиваем её в промпт. Это механизм **персонализации**: каждый тренер может «обучить» агента своей методике.

def get_recovery_prompt(athlete, fatigue, sleep, pain) -> str:

## Ключевые принципы написания промптов

1. **Структура** — задаём заголовки с `##`, LLM соблюдает структуру
2. **Конкретность** — «укажи числа, факты», «будь конкретным»
3. **Язык** — «Отвечай на русском языке» в конце каждого промпта
4. **Профиль** — всегда вставляем профиль спортсмена — контекст влияет на качество ответа
5. **Chain-of-Thought** — перечисляем что нужно сделать → модель думает структурированно
